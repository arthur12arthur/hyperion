# Architecture - Hyperion v5.0

Ce document décrit l'architecture générale du projet Hyperion v5.0 (version démo).
- Frontend : PWA statique (index.html, sw.js, manifest.json)
- Modules JS : logique client et micro-modules pour génération et distillation
- Backend optionnel : modules Python dans `python/`
- CI/CD : GitHub Actions (fichier simple dans .github/workflows)

Adapte cette architecture selon tes besoins réels.
