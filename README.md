# Hyperion v5.0

**Repository de démonstration pré-rempli** — préparé pour : `arthur12arthur/hyperion`

Ce dépôt contient une structure "production-ready" minimale et des scripts de démarrage.
Modifie les fichiers, ajoute ton code et adapte la configuration avant mise en production.

## Contenu principal
- Interface PWA (index.html, manifest.json, sw.js)
- Modules JS dans `js/`
- Modules Python dans `python/`
- Documentation dans `docs/`
- Scripts de build / déploiement dans `scripts/`
- CI/CD basique dans `.github/workflows/`

## Premiers pas
1. Dézippe l'archive et ouvre le dossier `hyperion`.
2. Ouvre `index.html` dans un navigateur pour voir l'interface.
3. Modifie `README.md` et `docs/` selon tes besoins.
4. Utilise `scripts/build.sh` puis `scripts/deploy.sh` pour builder et déployer.
